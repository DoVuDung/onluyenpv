# 11 — AI Features

- Generate Quiz (từ topic/company/difficulty)
- Generate Mock Interview (kịch bản hỏi-đáp theo role)
- Explain Wrong Answer (giải thích tại sao sai, kèm gợi ý học lại)
- Personal Learning Path (dựa lịch sử Attempt)
- AI Coach chat (context: hồ sơ + lịch sử làm bài)
- Code Review cho phần coding-challenge

## Nguyên tắc
- Mọi lời gọi LLM đi qua **1 AI Gateway module riêng** — không gọi thẳng SDK provider rải rác trong code — dễ đổi model, dễ theo dõi cost/log.
- Output của AI luôn được validate bằng Zod trước khi lưu DB hoặc trả về client (LLM có thể trả sai format).
- Job AI nặng (generate hàng loạt câu hỏi) chạy qua BullMQ, không block request HTTP.
- Cache kết quả AI theo input hash khi hợp lý để giảm chi phí.
